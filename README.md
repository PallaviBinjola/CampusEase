# CampusEase
CampusEase 🎓

CampusEase is a full-stack web application designed to modernize and automate essential campus services through a centralized digital platform. Built using Python (Flask), MySQL, HTML, CSS, and JavaScript, the system improves efficiency, reduces manual work, and enhances the overall campus experience for students, staff, and administrators.

🚀 Key Features
🍽️ Smart Canteen Pre-Ordering
Browse digital menus
Place orders in advance
Reduce waiting times and queues
Manage orders through a dedicated staff dashboard
🤖 AI-Based Food Recommendation
Recommends food items based on user preferences and order history
Uses machine learning techniques to provide personalized suggestions
Enhances user engagement and ordering experience
🔍 Intelligent Lost & Found System
Report lost or found items with images
Image-based matching using MobileNetV2
Similarity detection using Cosine Similarity
Helps users recover misplaced belongings faster
👥 Role-Based Access Control
Student Dashboard
Canteen Staff Dashboard
Administrator Dashboard
Secure authentication and authorization mechanisms
📊 Real-Time Analytics Dashboard
Monitor order trends and user activity
Track canteen performance
Visualize data through interactive charts and reports
Support data-driven decision-making for administrators
🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: MySQL
Machine Learning: TensorFlow/Keras, MobileNetV2, Cosine Similarity
Visualization: Chart.js
